About this dashboard
  • Model: Upload a scikit-learn classifier. If available, feature_names_in_ defines input order.
  • Dataset: CSV/XLS(X) with features. Add {{ target_name }} (0/1) to enable metrics.
  • Threshold: Controls the decision boundary for Abandoned vs Not Abandoned.
  • log1p: Applies log(1+x) to non-negative features to mirror training transforms.
  • Metrics: ROC-AUC, PR-AUC, PR/ROC curves, confusion matrix, and classification report.
  • Single prediction: Quick probability for one session using the same pipeline.
