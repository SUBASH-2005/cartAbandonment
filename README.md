# Predictive Cart Abandonment & Personalized Recommendation System

A machine learning–driven e-commerce solution that predicts cart abandonment and provides personalized product recommendations to improve customer engagement and sales conversion.

---

## 📌 Project Overview

This project focuses on identifying users who are likely to abandon their shopping carts and re-engaging them through intelligent, personalized product recommendations. By analyzing user behavior, session data, and interaction patterns, the system enables real-time interventions to reduce abandonment and increase conversions.

---

## 🚀 Features

- Predicts cart abandonment using machine learning models  
- Personalized product recommendations (content-based & collaborative filtering)  
- Real-time user behavior analysis  
- Flask-based web application  
- Scalable and modular architecture  

---

## 🧠 Technologies Used

- **Backend:** Python, Flask  
- **Machine Learning:** Random Forest, Gradient Boosting, RNN  
- **Recommendation System:** Collaborative & Content-Based Filtering  
- **Data Processing:** Pandas, NumPy  
- **Visualization:** Plotly  
- **Model Storage:** Pickle  
- **Frontend:** HTML, CSS (Flask Templates)
